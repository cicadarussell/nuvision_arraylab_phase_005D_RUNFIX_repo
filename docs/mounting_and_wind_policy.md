# Mounting and Wind Responsibility Policy

ArrayLab is a design-assist and workflow-accelerator system. It is not the final structural authority.

## Structural truth states

| State | Meaning | Behaviour |
|---|---|---|
| S0 | unknown | block install pack |
| S1 | precheck only | quote preview only |
| S2 | manufacturer calc required | export pack available |
| S3 | manufacturer calculated | report attached |
| S4 | engineer review required | block final pack |
| S5 | engineer approved | final evidence attached |

## ArrayLab may calculate

- roof edge/corner/middle zones
- approximate array distance from roof edges
- roof pitch compatibility
- roof type compatibility
- panel weight precheck
- mounting family suggestion
- thermal expansion split warnings
- missing survey data warnings

## ArrayLab must not claim

- final ballast approval
- final hook spacing
- final wind-load compliance
- final roof structural adequacy
- final weather-tightness
- final fire performance
- final MCS compliance
- final safe-to-install status

## Valk Assist pack

The tool prepares handoff data for Van der Valk / manufacturer / engineer review: address, roof outline, height, pitch, roof covering, module dimensions, layout coordinates, edge-zone overlay, shade report, proposed mounting family, preliminary BOM and warnings.
