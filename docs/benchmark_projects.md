# Benchmark Projects

Every release must pass benchmark projects before NuVision sees it.

| Benchmark | Why |
|---|---|
| simple south roof | baseline |
| east/west roof | split orientation |
| chimney shade roof | shade logic |
| awkward small roof | panel packing |
| edge-zone roof | wind/mounting warnings |
| trapezoidal commercial roof | mounting family mapping |
| flat roof ballast | manufacturer workflow |
| battery/hybrid install | inverter/battery matching |
| large C&I array | BOM/performance scale |
| old quote project | backwards compatibility |
